DirectX Screen Capture

Hardware-accelerated screen capture at 60 FPS using DirectX Desktop Duplication API with zero-copy GPU memory access for silky-smooth streaming.

Multi-threaded JPEG encoding with 4+ parallel encoder threads and bitmap pooling ensures maximum throughput while keeping CPU usage minimal and reducing memory pressure.

Dirty Region Optimization

Binary frame protocol with custom 64x64 tile-based dirty region tracking reduces bandwidth by 70-95% in static scenes through selective tile encoding.

Automatic keyframe insertion and frame skip detection ensure efficient streaming while maintaining visual quality, with bandwidth optimization for any network condition.

Gaming-Optimized Controls

Dual mouse modes with touchpad and direct positioning, virtual joystick with configurable WASD controls, and customizable action buttons designed specifically for gaming.

Full touch gesture support including two-finger scrolling, long-press for right-click, and drag-and-drop button positioning with resizable control elements and layout saving.
